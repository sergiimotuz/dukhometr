# ☦️ Духометр


**Православний духовний щоденник з AI-наставником та голосовою взаємодією**

![Духометр](https://img.shields.io/badge/Версія-1.0-purple) ![Ліцензія](https://img.shields.io/badge/Ліцензія-MIT-green) ![Мова](https://img.shields.io/badge/Мова-Українська-blue)

## 📖 Про додаток

**Духометр** - це ваш особистий православний духовний помічник, який допомагає:
- 📝 Вести щоденник духовного життя
- 🙏 Відстежувати духовні практики (молитви, піст, читання Євангелія)
- 💬 Спілкуватися з AI-наставником, навченим на творах святих отців
- 🎤 Використовувати голосову взаємодію
- 📊 Бачити свій духовний прогрес у статистиці
- ✨ Читати цитати святих отців

---

## ✨ Можливості

### 📖 Духовний щоденник
- Запис добрих вчинків та думок
- Фіксація гріхів для подолання
- Відстеження духовних практик
- Особисті роздуми

### 💬 AI-Наставник
- Мудрі відповіді на духовні запитання
- Знання святих отців (Іоанн Золотоустий, Серафим Саровський, Паїсій Святогорець)
- Цитати з Євангелія
- Практичні поради для духовного життя

### 🎤 Голосова взаємодія
- **Голосове введення** - говоріть питання голосом
- **Повна розмова** - AI відповідає голосом українською
- Три режими: текст, голосове введення, повна розмова

### 📊 Статистика
- Лічильники добрих вчинків і подоланих гріхів
- Графік активності за 7 днів
- Аналіз частоти духовних практик

### ✨ Мудрість святих
- Щоденна цитата від святих отців
- Бібліотека мудрих висловів

## 🚀 Як використовувати

### Онлайн версія
Просто відкрийте у браузері: **[посилання на ваш GitHub Pages]**

### Офлайн використання
1. Відкрийте додаток у браузері один раз
2. Додайте на головний екран телефону:
   - **iPhone/iPad:** Safari → Поділитися → "На головний екран"
   - **Android:** Chrome → Меню → "Додати на головний екран"
3. Додаток працюватиме офлайн!
