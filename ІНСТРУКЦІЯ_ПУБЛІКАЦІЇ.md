# 📖 Інструкція: Як опублікувати Духометр на GitHub Pages

## 🎯 Результат
Після виконання цих кроків ваш Духометр буде доступний за адресою:
`https://ваше-ім'я.github.io/duhometr`

---

## Крок 1: Реєстрація на GitHub

1. Перейдіть на **https://github.com**
2. Натисніть **"Sign up"** (Зареєструватися)
3. Введіть:
   - Email
   - Пароль
   - Ім'я користувача (це буде у вашій адресі сайту!)
4. Підтвердіть email

**💡 Порада:** Оберіть просте ім'я користувача, наприклад: `duhometr-ua`, `spiritual-app`, `pravoslavapp` тощо

---

## Крок 2: Створення репозиторію

1. Після входу натисніть **зелену кнопку "New"** (або "+")
2. Заповніть форму:
   - **Repository name:** `duhometr` (або інше ім'я)
   - **Description:** `Православний духовний щоденник з AI-наставником`
   - **Public** (має бути публічним!)
   - ✅ Поставте галочку: **"Add a README file"**
3. Натисніть **"Create repository"**

---

## Крок 3: Завантаження файлу

1. У вашому новому репозиторії натисніть **"Add file"** → **"Upload files"**
2. Перетягніть файл **`duhometr.html`** (який ви завантажили від мене)
3. Внизу сторінки натисніть **"Commit changes"**

**⚠️ ВАЖЛИВО:** Файл ОБОВ'ЯЗКОВО повинен називатися `index.html` (не `duhometr.html`)

Якщо файл називається `duhometr.html`:
- Натисніть на файл
- Натисніть іконку олівця (Edit)
- Змініть ім'я на `index.html`
- Збережіть (Commit changes)

---

## Крок 4: Налаштування GitHub Pages

1. У вашому репозиторії натисніть **"Settings"** (⚙️ Налаштування)
2. У лівому меню знайдіть **"Pages"**
3. У розділі **"Source"** оберіть:
   - Branch: **main** (або master)
   - Folder: **/ (root)**
4. Натисніть **"Save"**

---

## Крок 5: Отримання посилання

1. Зачекайте 1-2 хвилини
2. Оновіть сторінку
3. Вгорі з'явиться зелене повідомлення з посиланням:
   **"Your site is published at https://ваше-ім'я.github.io/duhometr"**

**🎉 Готово!** Тепер будь-хто може відкрити це посилання і користуватися Духометром!

---

## 📤 Як поділитися з іншими

Просто надішліть їм посилання:
```
https://ваше-ім'я.github.io/duhometr
```

Люди можуть:
- Відкрити в браузері
- Додати на головний екран телефону (працює як додаток!)
- Використовувати офлайн (після першого відкриття)

---

## 🔄 Як оновити додаток

Якщо я вам надішлю нову версію:

1. Відкрийте ваш репозиторій на GitHub
2. Клікніть на файл `index.html`
3. Натисніть іконку олівця ✏️
4. Скопіюйте весь новий код
5. Вставте замість старого
6. Натисніть **"Commit changes"**
7. Зачекайте 1-2 хвилини - сайт оновиться!

---

## ❓ Можливі проблеми

**Проблема:** "404 - Page not found"
- ✅ Перевірте, чи файл називається `index.html` (не `duhometr.html`)
- ✅ Перевірте, чи ввімкнені GitHub Pages
- ✅ Зачекайте 5 хвилин після публікації

**Проблема:** Голос не працює
- ✅ Це нормально - на деяких телефонах потрібні дозволи
- ✅ Спробуйте в Chrome або Safari
- ✅ Дайте дозвіл на використання мікрофона

**Проблема:** Дані не зберігаються
- ✅ Це нормально - дані зберігаються локально на пристрої
- ✅ Кожен користувач має свої власні дані
- ✅ При очищенні кешу браузера дані видаляються

## 🎁 Бонус: Власний домен (необов'язково)
