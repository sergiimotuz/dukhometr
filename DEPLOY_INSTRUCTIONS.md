# 🚀 Інструкція з деплою Духометра з AI на Vercel

## 📋 Що потрібно зробити (15 хвилин):

### Крок 1: Завантажити файли на GitHub

1. Перейдіть на https://github.com/sergiimotuz/dukhometr
2. Завантажте ці файли:
   - `api/chat.js` (в папку api/)
   - `package.json` (в корінь)
   - `vercel.json` (в корінь)
   - `index.html` (замініть існуючий)

**Структура має бути така:**
```
dukhometr/
├── api/
│   └── chat.js          ← новий файл
├── index.html           ← оновлений файл
├── package.json         ← новий файл
├── vercel.json          ← новий файл
└── README.md
```

### Крок 2: Підключити Vercel до GitHub

1. Зайдіть на https://vercel.com
2. Натисніть "Import Project"
3. Оберіть "Import Git Repository"
4. Оберіть `sergiimotuz/dukhometr`
5. Натисніть "Import"

### Крок 3: Додати API ключ Anthropic

1. В налаштуваннях проєкту Vercel знайдіть "Environment Variables"
2. Додайте змінну:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** ваш API ключ (починається з `sk-ant-...`)
3. Натисніть "Save"

### Крок 4: Задеплоїти

1. Натисніть "Deploy"
2. Зачекайте 1-2 хвилини
3. Vercel дасть вам URL типу: `https://dukhometr-xxxxx.vercel.app`

### Крок 5: Оновити URL в коді

1. Відкрийте `index.html`
2. Знайдіть рядок:
   ```javascript
   const response = await fetch('https://sergiimotuz-github-io-dukhometr.vercel.app/api/chat', {
   ```
3. Замініть на ваш Vercel URL:
   ```javascript
   const response = await fetch('https://ВАШ-URL.vercel.app/api/chat', {
   ```
4. Збережіть та закомітьте

Vercel автоматично передеплоїть!

## 🔑 Де взяти API ключ Anthropic?

### Варіант 1: Безкоштовні $5
1. Зареєструйтеся на https://console.anthropic.com
2. Підтвердіть email
3. Перейдіть в "API Keys"
4. Створіть новий ключ
5. Скопіюйте (починається з `sk-ant-...`)

**Це дасть вам:**
- ~1000 повідомлень безкоштовно
- Достатньо для тестування з 50 користувачами

### Варіант 2: Платний ($5-10/міс)
1. Додайте картку на https://console.anthropic.com
2. Поповніть на $5-10
3. Цього вистачить на 2000-4000 повідомлень

## ❓ Проблеми?

**Помилка "Ліміт вичерпано":**
- Це нормально - працює rate limiting
- Користувач зможе писати завтра

**Помилка "API key invalid":**
- Перевірте чи правильно скопіювали ключ у Vercel
- Ключ має починатися з `sk-ant-`

**AI не відповідає:**
- Перевірте Console в браузері (F12)
- Перевірте чи правильний URL API в `index.html`
